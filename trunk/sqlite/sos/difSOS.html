Very simple form to hit the DIF SOS

Target: http://carocoops.org/cgi-bin/microwfs/difSOS.cgi
OBSERVEDPROPERTY(required): listing of operator/observation counts available from the database are here
OFFERING(optional): like ndbc.41004.buoy, also accepts partial input for like comparison 'ndbc' for example will return all ndbc buoys. If unspecified will restrict data as specified by BBOX. Current offering list (organization/station/platform list) is available here
BBOX(optional): (min lon East, min lat North, max lon East, max lat North like -80.00,32.00,-75.00,36.00) in decimal degrees. If unspecified will restrict data as specified by OFFERING.
EVENTTIME(optional): (YYYY-MM-DDTHH:MM:SS/YYYY-MM-DDTHH:MM:SS like 2008-01-01T00:00:00/2008-12-31T23:00:00) in GMT. If unspecified will return latest observation. Available database keeps a maximum of the most recent 3 days data.
